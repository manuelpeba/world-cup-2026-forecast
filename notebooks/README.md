# Notebooks

This directory contains exploratory and analytical notebooks used during the development of the World Cup forecasting project.

These notebooks are **not required to run the simulation engine**.  
The production pipeline lives under `src/` and should be executed through the CLI or Python modules.

## Notebook Overview

### 00_eda_match_dataset.ipynb

Exploratory data analysis of the historical international match dataset used to build the modeling features.

Focus areas include:
- dataset structure
- target distribution
- temporal coverage
- feature sanity checks

### 02_simulation_analysis.ipynb

Analyzes the outputs of the World Cup tournament simulation.

This notebook focuses on understanding the structure of the forecast produced by the simulation engine. It examines how probabilities evolve across tournament stages and how championship likelihood is distributed across teams.

Key analyses include:

- Team progression probabilities (group stage → final)
- Championship probability distribution
- Forecast concentration and uncertainty
- Identification of top contenders and dark horses

The goal is to evaluate the behavior of the tournament simulation and interpret the probabilistic outputs generated by the forecasting pipeline.

### 03_world_cup_forecast_story.ipynb

Presents the results of the tournament simulation as a narrative forecast.

This notebook translates the probabilistic outputs of the model into a clear analytical story about the FIFA World Cup 2026 forecast.

It highlights:

- The main championship contenders
- Progression probabilities across tournament stages
- Contender tiers (favorites, challengers, dark horses)
- Forecast concentration and tournament uncertainty

Unlike the other notebooks, this notebook is designed primarily for **communication and storytelling**, presenting the results of the simulation in a format suitable for analytical reports.
